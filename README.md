### 一、相关技术

- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`

### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`


### 四、功能描述及功能图展示

系统分为：商城、管理员。

商城功能：1. 登录、2. 注册、3. 首页、4. 商品列表、5. 商品详情、6. 购物车、7. 历史订单

管理员功能：1. 登录、2. 订单管理、3. 商品管理
